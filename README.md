# IN Soluções Digitais — Site (V3)

Estrutura: **Empresa → Produtos → Serviços** (foco maior na empresa).

## Rodar local
```bash
npm install
npm run dev
```

## Onde editar
- Conteúdo/links (WhatsApp/instalador): `src/app/config.ts`
- Assets: `public/assets`

## Deploy
Domínio: `insolucoesdigitais.com.br`
